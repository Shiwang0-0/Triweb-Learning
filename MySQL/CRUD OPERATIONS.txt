------------------------------------------------------------------------------------------
DELETE only deletes record from the table,it cannot reset the AUTO INCREMENT value (means if 1,2,3 occupied as primary key number for a specific column and we deleted the whole table and then added another primary key number,it will start from 4) 
we have TRUNCATE for that.
------------------------------------------------------------------------------------------


------------------------------------------------------------------------------------------
TRUNCATE deletes the whole table not even leaving the auto incremented primary key numbers.
you cannot TRUNCATE a table refered in a foreign key constraint.
